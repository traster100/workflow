Скрипт чистки баз данных.

Порядок работы:

1. mysql.php
	- настройки сервера (хост-логин-пароль-бд)
	- настройки конфигурации (разрешенные\запрещенные таблицы\поля\типы полей)
    - настройка дебаг-режима (показывать конфигурацию таблиц\полей или нет)

2. select.php
	- массив ВЧ стоп-слов кладем в файл words1.txt
    - запускать лучше в консоли (форматирование под консоль)
	- генерируются и выполняются SELECT-запросы
	- успешные запросы складываются в файлы "имябд_слово.txt" в папку "results". каждое слово в свой файл. 
	- файлы потом открывать в notepad++ и собирать НЧ-слова в файл words2.txt. только слова! ссылку-родитель (если она есть) апдейтер порежет сам.

3. replaser.php
	- чистит поданное на вход значение
	- вырезает (если найдет) целиком тег <a> в котором слово, и все найденные слова вокруг
	- иначе вырезает просто слово

4. php update.php
	- забор из файла words2.txt собранных руками НЧ стоп-слов.
	- генерация и выполнение запросов SELECT, прогон через реплейсер, и затем UPDATE
