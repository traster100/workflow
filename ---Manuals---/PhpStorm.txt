Макрос "форматировать и сохранить"
1. В настройках Setting -> Keymap -> Save All - удалить сочетание клавишь Ctrl + S.
2. В меню Edit -> Macros -> Start Recording - и нажимаем Ctrl + Alt + L (форматирование) и потом Ctrl + S (сохранение). Тормозим запись. Edit -> Macros ->  Edit Macros и смотрим что записалось ReformatCode, SaveAll. Называем макрос reformat_save.
3. В настройках Setting -> Keymap -> reformat_save - ставим сочетание клавишь Ctrl + S.

Alt + F7 - поиск по всем местам проекта, в окне
Ctrl + Shift + F7 - подсветка всего в файле и перебор через F3
