Макрос "форматировать и сохранить"
1. В настройках Setting -> Keymap -> Save All - удалить сочетание клавишь Ctrl + S.
2. В меню Edit -> Macros -> Start Recording - и нажимаем Ctrl + Alt + L (форматирование) и потом Ctrl + S (сохранение). Тормозим запись. Edit -> Macros ->  Edit Macros и смотрим что записалось ReformatCode, SaveAll. Называем макрос reformat_save.
3. В настройках Setting -> Keymap -> reformat_save - ставим сочетание клавишь Ctrl + S.

Горячие клавиши

Ctrl+E - недавно открытые файлы
Ctrl+Shift+E - недавно редактированные файлы

Ctrl+F12 - навигация по структуре файла (нажать 2 раза чтобы увидеть суперклассы и интерфейсы)
Alt+Home - панель навигации по проекту

Ctrl+N - перейти в класс
Ctrl+Shift+N - перейти в папку или файл
Ctrl+Shift+Alt+N - перейти к символу

Alt+F1 - открыть файл в другом инструменте

Escape - фокус с таба на редактор
Shift+Escape - фокус с таба на редактор с закрытием таба

Ctrl+U - перейти в родительский класс или метод на иерархию выше
Ctrl+Alt+B - перейти в дочерний класс или метод на иерархию ниже

Ctrl+Shift+H - иерархия метода

Ctrl+H - иерархия класса
Ctrl+Alt+U - диаграмма иерархии класса

Ctrl+Alt+H - иерархия вызовов метода

Alt+Ctrl+F7 - Показать использования символа/переменной/константы/методы/класса

Ctrl+Alt+Home - навигация основанная на определенных отношениях типа контроллер-представление

F11 - закладки
Shift+F11 - выбрать закладку

Ctrl+Shift+Alt+T - покажет все рефакторинги в данном месте
Alt+Enter - покажет все инспекции в данном месте
