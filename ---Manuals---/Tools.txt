Chromium

- скачать отсюда https://chromium.woolyss.com/#windows-64-bit

- файл Chromium-78.0.3904.70-x64, в списке выглядело так:
"Portable • Archive • Installer • Package → Marmaduke thinLTO polly widevine all-codecs sync win64"

- скачать тип Archive, а не Portable

- флаг sync - версия с API-ключами для авторизации в гугле

- смена путей до профиля, и папки кеша:
1. смотрим тут chrome://version
2. делаем ярлык приложения и там пишем:
G:\soft\Chromium\chrome.exe --user-data-dir=G:\soft\Chromium\Profile --disk-cache-dir=C:\Users\u\AppData\Local\Temp\ChromiumCache --disable-reading-from-canvas

флаг --disable-reading-from-canvas нужен для анонимности. проверка тут:
https://browserleaks.com/canvas
https://panopticlick.eff.org

Photoshop. Создание макроса

1. Меню Window - Actions.
2. В палитре нажать Create new action, задать имя, нажать Record.
3. Провести все процедуры, сохранить картинку, нажать Stop Record.
4. File - Automate - Batch, выбрать макрос, выбрать папки folder_from и folder_to, нажать ОК.

PhpStorm

1. Поставить галки:

Размер шрифта менять колесом мыши
Editor->General->Change font size (Zoom) with Ctrl+MouseWheel

Перенос текста в консоли
Editor->General->Console->Use soft wraps in console

Перенос текста в редакторе
Editor->General->Soft-wrap files *.*

Горизонтальная линия между методами, в классе
Editor->General->Appearance->Show method separators

2. В файле \bin\idea.properties:

Профиль в папке PhpStorm
idea.config.path=${idea.home}/.PhpStorm/config
idea.system.path=${idea.home}/.PhpStorm/system

Удалить задержку IDE при печатании
editor.zero.latency.typing=true

3. В файле \bin\phpstorm64.exe.vmoptions:

Увеличенный кеш
-Xms1024m
-Xmx2048m
-XX:MaxPermSize=1024m

Кряк
-javaagent:crack.jar

Сглаживание шрифтов
-Dawt.useSystemAAFontSettings=lcd

Аппаратное ускорение
-Dawt.java2d.opengl=true

Отключение поддержки мониторов HiDPI
-Dsun.java2d.uiScale.enabled=false

3. Отключить обновления
Appearance & Behavior->System Settings->Updates->Automatically check updates for

ДАЛЕЕ РАЗБИРАТЬ

Макрос "форматировать и сохранить"
1. В настройках Setting -> Keymap -> Save All - удалить сочетание клавишь Ctrl + S.
2. В меню Edit -> Macros -> Start Recording - и нажимаем Ctrl + Alt + L (форматирование) и потом Ctrl + S (сохранение). Тормозим запись. Edit -> Macros ->  Edit Macros и смотрим что записалось ReformatCode, SaveAll. Называем макрос reformat_save.
3. В настройках Setting -> Keymap -> reformat_save - ставим сочетание клавишь Ctrl + S.
